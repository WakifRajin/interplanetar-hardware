| AS5600 Pin | Description                                         |
| ---------- | --------------------------------------------------- |
| `VDD`      | Power supply (3.3V or 5V)                           |
| `GND`      | Ground                                              |
| `SDA`      | I²C data line                                       |
| `SCL`      | I²C clock line                                      |
| `DIR`      | Direction reference (GND = normal, HIGH = reversed) |


install the library:
Sketch > Include Library > Manage Libraries > Search “AS5600” by Rob Tillaart (or similar)
